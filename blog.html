    weebly

    Weebly is more of a user-friendly web-hosting service because of its drag-and-drop features it's really good for beginners and for anyone who wants to make a quick website , but it does have its limitations, Weebly doesn't give you give the user a lot of customization abilities it seems like if a person wants to make a more customised and unique website they have to pay for Weebly's premium addition which gives the user a lot more tools to work with. So overall its an ok website maker but there is another website maker that is a lot better.
